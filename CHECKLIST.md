## 📋 Проверка data-атрибутов (для автотестов)

Откройте DevTools (F12) и проверьте селекторы:

**На главной странице:**
```javascript
// Должны быть элементы с этими атрибутами
document.querySelector('[data-title]')          // Название товара
document.querySelector('[data-price]')          // Цена товара
document.querySelector('[data-registration]')   // Форма регистрации
```

**На странице корзины:**
```javascript
document.querySelector('[data-title="basket"]') // Название в корзине
document.querySelector('[data-price="basket"]') // Цена в корзине
document.querySelector('[data-delivery]')       // Форма доставки
```

## 🐛 Если что-то не работает

### Бэкенд не запускается
```bash
# Проверьте порт 3000
netstat -ano | findstr :3000  # Windows

# Пересоберите бэкенд
npm run build
```

### Фронтенд не загружается
```bash
# Пересоберите фронтенд
npm run frontend:build

# Проверьте, что bundle.js создан
ls frontend/bundle.js
```

### Корзина не работает
- Убедитесь, что вы авторизованы
- Проверьте console в DevTools на ошибки
- Очистите cookies браузера

### CORS ошибки
- Убедитесь что бэкенд на http://localhost:3000
- Убедитесь что фронтенд на http://localhost:5500
- Перезагрузите браузер
